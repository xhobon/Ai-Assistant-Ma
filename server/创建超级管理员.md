# 创建超级管理员账号

## 方法一：使用环境变量（推荐）

1. 在 `.env` 文件中添加以下配置：
```env
SUPER_ADMIN_EMAIL=admin@ai-assistant.com
SUPER_ADMIN_PASSWORD=SuperAdmin2026!
SUPER_ADMIN_NAME=超级管理员
```

2. 运行创建脚本：
```bash
npm run create-super-admin
```

## 方法二：使用默认值

直接运行脚本，将使用默认值：
- 邮箱：`admin@ai-assistant.com`
- 密码：`SuperAdmin2026!`
- 名称：`超级管理员`

```bash
npm run create-super-admin
```

## 默认超级管理员账号

- **邮箱**: `admin@ai-assistant.com`
- **密码**: `SuperAdmin2026!`
- **角色**: `super_admin`

⚠️ **重要提示**：
- 首次登录后请立即修改密码
- 请妥善保管超级管理员账号信息
- 不要在生产环境中使用默认密码

## 管理员角色说明

- **user**: 普通用户（默认）
- **admin**: 管理员
- **super_admin**: 超级管理员（最高权限）
