# 最后一步：在终端执行部署（约 1 分钟）

数据库已就绪，环境变量已写在 `server/.env`。你只需在**本机终端**执行下面这一条命令完成上线。

## 执行（复制整段到终端）

```bash
cd "/Users/akun/Desktop/Ai助理/Ai助理/server"
npm install
npx vercel --prod
```

- **第一次**会弹出浏览器，用 GitHub/邮箱登录 Vercel；
- 若提示 **“Link to existing project?”** 选 **N**；
- 若提示 **“What’s your project’s name?”** 填：**ai-assistant-backend**（这样 App 内置地址才能对上）；
- 之后等待部署完成，会看到一行 **Production: https://ai-assistant-backend-xxx.vercel.app**。

## 部署完成后

1. 在 Vercel 网站里给项目加上环境变量（若部署时报错再补）：
   - 打开 https://vercel.com 你的项目 → **Settings** → **Environment Variables**
   - 添加 3 个（Name / Value）：
     - `DATABASE_URL` → 你的 Neon 连接串（和 .env 里一致）
     - `JWT_SECRET` → `XHOBON_2026_SuperSecretKey_9xK3LpA8mZ`
     - `DEEPSEEK_API_KEY` → `sk-2a523738a2664059a63dd0584eca7361`
   - 保存后到 **Deployments** 里点 **Redeploy** 一次。

2. App 已内置默认地址 **https://ai-assistant-backend.vercel.app**。  
   若你实际得到的域名是 `https://ai-assistant-backend-你的用户名.vercel.app`，把**完整地址**发给我，我会把 App 里的默认地址改成这个。

3. 打开 App，直接使用翻译 / AI 对话 / 语音通话即可，**无需在设置里填服务器地址**。
